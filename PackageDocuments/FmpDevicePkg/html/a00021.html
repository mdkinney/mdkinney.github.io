gFmpDevicePkgTokenSpaceGuid.PcdFmpDeviceProgressColor

The color of the progress bar during a firmware update. Each firmware
device can set its own color. The default color is white.

Bits 15..8 - Green<BR>
Bits 23..16 - Blue<BR>
